# AgriDoc వినియోగదారు మాన్యువల్ | User Manual

## పరిచయం | Introduction

AgriDoc మీ వ్యవసాయ పత్రాలను సురక్షితంగా నిల్వ చేయడానికి సహాయపడుతుంది.
AgriDoc helps you store your farm documents safely — even without internet.

---

## 1. ప్రారంభించడం | Getting Started

### ఇన్‌స్టాలేషన్ | Installation

**సాధారణ పద్ధతి | Simple method:**
```bash
pip install -e .
python run.py
```

బ్రౌజర్‌లో తెరవండి: http://localhost:5000

### Docker:
```bash
docker run -p 5000:5000 agridoc
```

---

## 2. డాష్‌బోర్డ్ | Dashboard

హోమ్ పేజ్‌లో మీకు కనిపిస్తుంది:
- **మొత్తం పత్రాలు** — మీరు నిల్వ చేసిన పత్రాల సంఖ్య
- **రైతులు** — నమోదు చేసిన రైతుల సంఖ్య
- **ధృవీకరించబడిన** — వెరిఫై అయిన పత్రాలు
- **సింక్ పెండింగ్** — అప్‌లోడ్ కావలసిన పత్రాలు

---

## 3. పత్రం అప్‌లోడ్ చేయడం | Uploading a Document

1. "అప్‌లోడ్" బటన్ నొక్కండి (పైన నారింజ రంగు బటన్)
2. ఫైల్ ఎంచుకోండి (PDF, JPG, PNG) — లేదా డ్రాగ్ అండ్ డ్రాప్ చేయండి
3. పత్రం పేరు టైప్ చేయండి
4. పత్రం రకం ఎంచుకోండి (పట్టాదార్, ఆధార్, రైతు బంధు...)
5. రైతు పేరు ఎంచుకోండి (ఐచ్ఛికం)
6. "పత్రం సేవ్ చేయండి" నొక్కండి

### మద్దతు ఉన్న ఫైల్ రకాలు | Supported File Types
- PDF (.pdf)
- చిత్రాలు (.jpg, .jpeg, .png, .webp)
- గరిష్ట పరిమాణం: 16 MB

---

## 4. పత్రాలు వెతకడం | Searching Documents

1. "పత్రాలు" మెనుపై క్లిక్ చేయండి
2. శోధన పెట్టెలో పత్రం పేరు లేదా రైతు పేరు టైప్ చేయండి
3. రకం ఫిల్టర్ ఉపయోగించి నిర్దిష్ట పత్రాలు వెతకండి

---

## 5. ఆఫ్‌లైన్ వినియోగం | Offline Usage

AgriDoc ఇంటర్నెట్ లేకుండా పని చేస్తుంది:
- మీ పత్రాలు మీ పరికరంలో నిల్వ అవుతాయి
- ఆఫ్‌లైన్ బ్యానర్ కనిపిస్తే — "మీరు ఆఫ్‌లైన్‌లో ఉన్నారు"
- ఇంటర్నెట్ వస్తే స్వయంచాలకంగా సింక్ అవుతుంది

## 6. పత్రం రకాలు | Document Types

| పత్రం | వర్గం | వివరణ |
|--------|--------|--------|
| పట్టాదార్ పాస్‌బుక్ | భూమి | భూ యాజమాన్య హక్కు |
| అదంగల్ / RoR | భూమి | Rights of Record |
| రైతు బంధు | పథకం | పెట్టుబడి మద్దతు |
| పీఎం-కిసాన్ | పథకం | PM-KISAN పథకం |
| పంట బీమా | బీమా | Crop Insurance |
| ఆధార్ కార్డ్ | గుర్తింపు | Identity Proof |
| బ్యాంక్ పాస్‌బుక్ | ఫైనాన్స్ | Banking |
| మట్టి ఆరోగ్య కార్డు | వ్యవసాయం | Soil Health Card |

## 7. సమస్యలు & పరిష్కారాలు | Troubleshooting

**యాప్ ప్రారంభం కాలేదు:**
```bash
pip install -e .
python run.py
```

**పత్రం అప్‌లోడ్ కాలేదు:**
- ఫైల్ పరిమాణం 16MB కంటే తక్కువగా ఉండాలి
- PDF, JPG, PNG మాత్రమే మద్దతు ఉంది

**సహాయం కోసం:** GitHub Issues లో రిపోర్ట్ చేయండి.
